              # 新手上路

              # 下载

              下载并安装 了 GameAI.win 之后，你需要注册一个帐号。

              # 运行

              • 双击 GameAI.win.exe 启动

              # 登陆

              • 您现在可以使用测试帐号test密码test登陆

              # 中控台


              上线 下线 暂停脚本 重启脚本 定时关闭

              • 橙色表示主机暂停，绿色表示工作，灰色表示被禁用。

              # 添加主机

              1. 点击 + 按钮
              2. 输入主机名称，主机IP，端口，用户名，选择脚本
              3. 点击右上角保存图标，保存主机信息

              # 主机配对

              # XBOX

              • 无须配对，可直接 连接到主机

              # PlayStation

              1. 点击主机列表中的想要上线主机的上线图标
              2. 如果第一次连接，需要进行主机配对
              3. 当需要进行主机配对时，请手动操作主机到达配对页面，查看配对码
              4. 输入配对码进行配对
              5. 成功连接到主机后，会显示投屏画面

              # Switch

              1. 即将加入

              # 连接到主机


              上线 下线 暂停脚本 重启脚本 定时关闭

              1. 点击对应主机选项卡上的 上线 按钮即可连接到主机

              # 运行脚本

              1. 主机连接成功后会自动启动选中的脚本
              2. 如果要更换脚本，请直接在主机列表中点击对应主机，选择脚本，保存 即时生效。

              # 自定义脚本?

              1. 如果您需要自定义脚本，请参考 脚本编写
